# BotFlix - Premium Cinematic Application
---

## Key Features
---
*   **TMDB API Integration**: Dynamically fetches the latest movies, trending series, and popular titles from The Movie Database (TMDB).
*   **Intelligent Category Filtering**: Features organized rows for Trending, Action, Comedy, Horror, and Top Rated content for quick navigation.
*   **Cinematic Detail View**: Provides immersive hero sections and detailed movie modals including synopses, ratings, and high-resolution posters.
*   **Interactive UI Controls**: Includes a "Watch Now" interaction model and smooth cinematic transitions for enhanced user engagement.
*   **Visual Excellence**: Designed with a high-contrast dark theme, glassmorphic navigations, and fluid hover animations.
*   **Responsive Architecture**: Fully optimized for desktop, tablet, and mobile devices to ensure a premium viewing experience everywhere.

## Technologies Used
---
*   **Frontend**: HTML5, Tailwind CSS, Google Fonts
*   **API**: The Movie Database (TMDB) API
*   **Logic**: Vanilla JavaScript (ES6)
